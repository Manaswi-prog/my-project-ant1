# Pathwise AI Architecture

## Overview
Pathwise AI is a vanilla JavaScript client-side application heavily reliant on Gemini 2.0 Flash for generating personalized career roadmaps and Firebase for session analytics tracking.

## Components
- src/main.js: Controller for interaction, data orchestration, and UI updates.
- src/careerData.js: Fallback static routing engine for offline/fallback intelligence.
- src/firebase.js: Firebase backend integrations.
- src/particles.js: Canvas-based performant background particle generation.
- src/cursor.js: Global custom cursor replacement logic.
