## 🧠 Approach & Logic

1. **Frictionless Voice & NLP Form:** The user is greeted by a cinematic 3D Next-Gen UI. They can use their native microphone (`🎙️`) to input free-text data describing their professional state without typing.
2. **Analysis Stage (The Hacker Terminal):** As data ships to **Google Gemini 2.0 Flash**, the DOM shifts into a custom-styled pulsing hacker terminal, enhancing UX during API resolution latency. 
3. **Execution Stage:** Gemini calculates match probability, synthesizes core gaps (Low, Med, High priority), and paces a step-by-step roadmap tailored to the exact months available.
4. **Presentation:** The UI parses the strict JSON response, animates the 2D Canvas Radar chart calculating vectors mathematically based on the AI feedback, and securely appends the session log to Firebase.

- **Code Quality:** Pure Vanilla Javascript architecture. Zero framework overhead (Under `1 MB` total repository footprint!). 100% fixed ESLint execution.
